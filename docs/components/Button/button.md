## 图标按钮

带图标的按钮可增强辨识度（有文字）或节省空间（无文字）。

 <div class="flex flex-row">
    <CButton icon="edit" plain></CButton>
    <CButton icon="delete" plain></CButton>
    <CButton icon="share" plain></CButton>
    <CButton round plain icon="search">搜索</CButton>
 </div>
